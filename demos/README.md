## How These Were Made

```
muser
> Compose a lo-fi hip hop track called "Sunday Morning" with dusty piano,
> warm vinyl crackle, and a lazy Sunday feel
```

The Muser plans the composition, generates candidates via ACE-Step v1.5,
scores each with the 9-metric quality analyzer, selects the best, and
exports production-ready audio.

All output is commercially safe (Apache 2.0 licensed models, no CC-BY-NC-SA
components in the default pipeline).
