# Roadmap

## Planned Features

### Near Term
- **HuggingFace Spaces demo** — try The Muser in the browser without installing
- **fish-hifigan integration** — verified Apache 2.0 vocoder for DiffSinger
- **Session fine-tuning pipeline** — use logged sessions to improve orchestration quality
- **Expanded voice training** — streamlined pipeline from recording to trained LoRA

### Medium Term
- **MIDI-LLM integration** — language models trained on MIDI token sequences for direct symbolic generation
- **Real-time audio preview** — stream audio during generation instead of waiting for completion
- **Multi-user sessions** — collaborative composition via shared state
- **Plugin system** — extensible tool registration for community-contributed generators

### Long Term
- **Cloud-hosted inference** — optional paid tier for users without GPUs
- **Mobile companion app** — review and edit compositions on the go
- **DAW integration** — VST/AU plugin for use inside Ableton, Logic, etc.
- **Video scoring** — compose music synchronized to video timecodes
